Project Description

This project is a Retrieval-Augmented Generation (RAG) chatbot designed to help employees find answers from internal documents and meeting transcripts. The system focuses on secure access, consistent retrieval, and clear responses backed by the knowledge base content.

Skills Learned

  • RAG concepts: retrieval + generation working together
  • Building ingestion pipelines for documents and transcripts
  • Structuring an application for maintainability (modular components)
  • Thinking about security and access control for internal tools
  • Communicating technical results to non-technical stakeholders
